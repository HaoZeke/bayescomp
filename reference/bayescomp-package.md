# bayescomp: Bayesian Hierarchical Comparison of Computational Method Benchmarks

A targets-integrated workflow for fitting Bayesian hierarchical models
to computational method benchmarks. Supports negative binomial (count
data), Gamma (timing), Bernoulli (success), and Student-t (signed
errors) families with random intercepts per test system. Provides
automated family selection, weakly informative prior defaults,
convergence diagnostics, LOO-CV, posterior predictive checks, and
publication-quality effect summaries. Built on 'brms' and 'cmdstanr'.
